- MongoDB

## Tech stack

- Frontend
  - React
  - React Router
  - [Shoto UI](https://shotoui.netlify.app)
- Backend
  - Express
  - Mongoose
  - MongoDB
  - JWT

## Features

- User authentication : Login/SignUp
- Create/Edit/Delete playlists
- Add/Remove videos from playlist
- Subscribe/Unsubscribe from categories
